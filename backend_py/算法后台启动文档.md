# 算法后台启动文档

安装uv

```JSON
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```





1、在backend_py目录下，安装依赖

使用requirements.txt安装依赖：
输入指令：

```Plain
uv sync
```

2、启动虚拟环境

```JSON
.venv\Scripts\Activate.ps1
```

3、配置.env文件

```Plain
cp .env.example .env
```

修改.env文件中的配置

4、在app目录下，运行项目

```JSON
cd app
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

5、可访问的接口文档

```Plain
http://127.0.0.1:8000/docs
```
